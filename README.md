## 📖 Sobre o Projeto

Este projeto consiste na construção de um **formulário de matrícula**, desenvolvido com foco em:

* organização de campos e inputs
* estrutura semântica
* fidelidade ao layout do Figma
* experiência do usuário (UX)

O objetivo foi praticar a criação de formulários completos e bem estruturados utilizando apenas **HTML e CSS**.

---

## 🎯 Objetivo

* Criar um formulário funcional e organizado
* Trabalhar diferentes tipos de inputs
* Aplicar boas práticas de HTML semântico
* Reproduzir fielmente um layout do Figma

---

## 🧠 Principais Aprendizados

Durante o desenvolvimento, foram trabalhados:

* Estruturação de formulários com HTML (`form`, `input`, `label`)
* Uso de diferentes tipos de inputs
* Organização visual de formulários
* Estilização com CSS
* Alinhamento e espaçamento de elementos
* Fidelidade ao design (pixel perfect)
